# The Professional Courier Viman Nagar Website

This repository contains a static website for **The Professional Courier Viman Nagar** branch.

## Included Documents and Pages

- `index.html`: Home page
- `services.html`: Service listing
- `tracking.html`: Shipment tracking form (demo interaction)
- `about.html`: Branch overview and coverage area
- `contact.html`: Contact details and callback request form
- `privacy-policy.html`: Privacy policy document
- `terms.html`: Terms and conditions document
- `robots.txt`: Search crawler policy
- `sitemap.xml`: Search indexing map
- `manifest.webmanifest`: Web app metadata

## Run Locally

Open `index.html` in your browser.

For local server preview in VS Code terminal:

```powershell
python -m http.server 8080
```

Then open <http://localhost:8080>.

## Customization Notes

- Update contact details in `contact.html`.
- Replace placeholder tracking logic in `script.js` with real API integration.
- Add brand assets to `assets/images/`.
- For homepage visuals, add these files in `assets/images/`:
	- `professional-courier-banner.jpg` (main top banner image)
	- `owner-shashi-singh.jpg` (owner section portrait image)
